WCF

CONCEITOS BÁSICOS

  • • Foi criado a partir do .NET Framework 3.0
  • • Unificou as várias tecnologias de programação distribuídas na plataforma Microsoft em um único modelo
  • • Baseou-se na arquitetura orientada a serviços (SOA)

WCF

CONCEITOS BÁSICOS

  • • Seu principal objetivo é permitir que analistas e desenvolvedores criem aplicações voltadas para computação distribuída.
  • • Possui ainda um conjunto de bibliotecas (classes) que permitem aos desenvolvedores criar estas aplicações para funcionarem sob o sistema operacional Windows.

WCF

PRINCIPAIS VANTAGENS

  • • Fornece maior confiabilidade e segurança em relação às tecnologias anteriores.
  • • Melhora o tráfego de informações pela rede, a fim de diminuir informações desnecessárias.
  • • Flexibiliza e aumenta as possibilidades de interação dos serviços com outros sistemas.

Front-end

O papel do desenvolvedor front-end

Ele é o responsável por projetar, construir e otimizar as interfaces de projetos web

Responsabilidades:

  • 1. Assegurar compatibilidade cross-browser e cross-platform
  • 2. Produzir código limpo, bem documentado e estruturado
  • 3. Seguir os web standards

Front-end

Importância da engenharia front-end

  • 1. 80% do tempo de carregamento é gasto com o download dos elementos front-end (css, js, imagens etc)
  • 2. As interfaces das aplicações estão cada vez mais complexas
  • 3. Solucionar problemas de cross-browsers
